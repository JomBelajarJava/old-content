Semasa membuat deployment untuk program JavaFX, jika anda perasan selain daripada deploy sebagai desktop application, kita juga boleh deploy sebagai web application. Walaupun boleh, saya tidak sentuh langsung mengenai web application menggunakan JavaFX. Ini adalah disebabkan oleh masalah platform untuk web application itu sendiri, iaitu browser (seperti Google Chrome dan Mozilla Firefox).

Program Java menggunakan teknologi Java WebStart untuk run di browser. Browser memerlukan Java plugin untuk melancarkan Java WebStart. Masalah berlaku apabila browser tersebut tidak lagi membenarkan Java plugin atas sebab-sebab security. Masalah yang sama berlaku pada Flash apabila Mozilla Firefox tidak lagi membenarkan penggunaan Flash di browser tersebut. Oleh itu, teknologi DukeScript dilancarkan untuk mengatasi masalah ini.

Website biasanya menggunakan HTML5+CSS3+JavaScript, manakala DukeScript pula menggantikan JavaScript tersebut dengan Java. Jadi, secara ringkasnya DukeScript boleh diibaratkan sebagai HTML5+CSS3+Java.

Sebelum ini kita lihat untuk run program Java, kita perlu menggunakan Java Runtime Environment (JRE) yang kita perlu download dan install. Untuk DukeScript pula, kita menggunakan JRE yang lain, antaranya TeaVM dan Bck2brwsr. TeaVM dan Bck2brwsr ditulis menggunakan JavaScript jadi kita tidak perlu risau sama ada user mempunyai Java ataupun tidak.

Selain untuk web application, DukeScript juga boleh target untuk mobile (seperti Android dan iOS) dan desktop.
